# 已阅读论文概要

## Toolformer: Language Models Can Teach Themselves to Use Tools

- 年份：2023
- 方向：大语言模型、工具调用
- 阅读状态：已完成
- 主要内容：研究语言模型如何通过自监督方式学习何时调用外部工具、选择什么工具，以及如何把工具返回结果用于后续文本生成。
- 核心价值：说明语言模型可以在生成过程中主动使用计算器、搜索、翻译等外部工具，从而弥补参数知识和计算能力的不足。
- 与当前研究的关系：可为 CottonBrain 中的工具选择、工具调用和工具结果融合提供方法依据

## API-Bank: A Comprehensive Benchmark for Tool-Augmented LLMs

- 年份：2023
- 方向：大语言模型、工具调用评测
- 阅读状态：已完成
- 主要内容：API-Bank构建了一个面向工具增强大模型的训练与评测基准，使用可执行API和人工标注数据评价模型的API调用能力，并通过五智能体流程生成大规模训练数据。它将能力划分为Call、Retrieval+Call和Plan+Retrieval+Call三个层级。
- 核心价值：将工具调用过程与最终回答分开评价，能够判断模型是否真正完成了正确的工具选择、参数生成、检索和多步规划。
- 与当前研究的关系：可用于设计CottonBrain的分层工具调用任务、标准调用轨迹和过程性评价指标。
